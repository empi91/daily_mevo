## Notes

GitHub issue #36 [E-14]. When a user opens a station page, the "Morning" tab is always expanded by default regardless of the actual time. The tab matching the current part of the day (morning / afternoon / evening / night) should be auto-selected on load so users immediately see the data most relevant to them.
